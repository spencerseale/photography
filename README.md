# Photography Website

Personal photography portfolio website powered by Jekyll and Flickr.

## Installation

Install dependencies:

```bash
gem install bundler
bundle install
```

## Development

Start the local development server:

```bash
bundle exec jekyll serve
```

Visit `http://localhost:4000` to view the site.

## Configuration

Update your Flickr credentials in `_config.yml`:

```yaml
flickr_user_id: "your_flickr_user_id"
flickr_username: "your_flickr_username"
```
